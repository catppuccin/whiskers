You can install whiskers using any method listed below.

## Source

```shell
cargo install --git https://github.com/catppuccin/whiskers
```

## Binaries (Windows, macOS & Linux)

Available from the
[latest GitHub release](https://github.com/catppuccin/whiskers/releases).

## crates.io

```shell
cargo install catppuccin-whiskers
```

## Homebrew

```shell
brew install catppuccin/tap/whiskers
```

## Nix/Nixpkgs

Available in
[nixpkgs](https://search.nixos.org/packages?channel=25.05&show=catppuccin-whiskers&query=catppuccin-whiskers)
as `catppuccin-whiskers`.

```shell
nix-shell -p catppuccin-whiskers
```

## AUR

Install [catppuccin-whiskers-bin](https://aur.archlinux.org/packages/catppuccin-whiskers-bin)
